Introduction

DynamicPDF for .NET includes DynamicPDF ReportWriter for .NET, DynamicPDF Merger for .NET and DynamicPDF Generator for .NET.

DynamicPDF Designer

The DynamicPDF Designer is a layout tool that allows you to define template for the ReportWriter reports using WYSIWYG controls. This template is saved as a DPLX file, which is then used in the ReportWriter to create the PDF reports. It comes along with DynamicPDF CoreSuite for .NET or DynamicPDF ReportWriter for .NET.

How to create DPLX files using Designer is explained in the User manual. But in the examples folder some report writer examples shows how to use the DPLX files in the Report Writer.

DynamicPDF ReportWriter for .NET

DynamicPDF ReportWriter for .NET is based on our popular DynamicPDF ReportWriter for COM/ActiveX product. It makes the process of creating real-time PDF reports simple without sacrificing performance. It easily integrates with DynamicPDF Merger for .NET, allowing for pages from an existing document to be prepended and appended to a report. DynamicPDF Generator for .NET is included with DynamicPDF ReportWriter for .NET.

DynamicPDF Generator for .NET

DynamicPDF Generator for .NET is a highly efficient class library used for the generation of PDF documents based on dynamic data. PDF documents can be generated straight to a file, a web page or to an output stream object. The object model is intuitive and easy to learn, yet at the same time is very powerful.

DynamicPDF Merger for .NET

DynamicPDF Merger for .NET makes simple the task of merging and adding new content to existing PDF documents. The Merger object model is intuitive and easy to learn yet very flexible allowing PDF merging, stamping, appending, form filling and page placing, rotating and scaling. DynamicPDF Generator for .NET is included with DynamicPDF Merger for .NET.

Binaries

.NET Assemblies

There are three .NET assemblies included that are compiled for use with the .NET Framework 2.0, 3.0, 3.5, 4.0 and 4.5. The table below gives the details of each assembly.

Assembly Name Assembly Version Framework Version
ceTe.DynamicPDF.20.dll 7.0.2.20 2.0 and 3.0
ceTe.DynamicPDF.35.dll 7.0.2.35 3.5
ceTe.DynamicPDF.40.dll 7.0.2.40 4.0 and 4.5
These are 100% managed assemblies written in C# and use only safe code. They are installed by default in the bin\ folder. The accompanying xml files are for intelli-sense and do not need to be distributed with your product.

The "DpdfDesigner.exe" executable is the DynamicPDF Designer application. It is installed by default in the root folder.
